Generate Move Labels

The move labels tool allows you to generate a set of Avery formatted move labels you can attach to moving boxes!

Simply export an Excel/XLSX move report from OfficeSpace, upload your moves .XLSX file and the tool generates an Avery formatted PDF file you can use to print your move labels.

Instructions

  1. Within OfficeSpace Move reports, generate a new move report and export/download the report to your computer. (XLSX format).
  2. Configure your labels per the following settings:
    1. Avery Template  Specifies the Avery template format you'd like to print to. We currently support 5160, 9400/94207 templates.
    2. Number of Labels per Move  Indicates the number of labels to create for each move in the report/Excel file. For example, if you specify "4", the Avery PDF output file will include 4 labels for each move.
  3. Click or tap "Upload Moves XLSX File" to upload and start the template generation process. Your Avery formatted PDF file will automatically download.
  4. Load your printer with the same Avery template you chose. Print your Avery formatted PDF file.
  5. Peel and stick to boxes, chairs and more!
